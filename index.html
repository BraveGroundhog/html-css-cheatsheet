HTML/CSS Reference

HTML Tags

Tag Name Description
<h1>-<h6> Header 1-6 Page heading 1-6, 1 being the highest level and 6 being the lowest.
<p> Paragraph Block of text forming the body of a website's text.
<div> Divider/Division Creates divisions between sections of html code. Can have class tags added to further divide.
<ul> Unordered List Creates a bulleted list, must have list item elements nested.
<ol> Ordered List Creates a numbered list, must have list item elements nested.
<li> List Item Forms the items in a list. Must be nested in <ul> or <ol> in order to be displayed.
<a> Anchor Allows for insertion of a link to a web page. Can be used to turn most elements into a link.
